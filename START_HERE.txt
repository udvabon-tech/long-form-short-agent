🎬 What is this?

A complete, automated pipeline to transform long Bengali podcasts
into viral-worthy vertical reels with:
  • AI-powered transcription (Gemini 2.5 Pro)
  • Word-by-word karaoke subtitles
  • Professional Bengali title overlays
  • Mobile-optimized 608x1080 format

⚡ Quick Setup (30 seconds)

1. Install dependencies:
   pip install -r requirements.txt

2. Get Gemini API key:
   https://aistudio.google.com/apikey

3. Set API key:
   export GEMINI_API_KEY="your-key-here"

4. Start creating:
   Follow QUICKSTART.md

🎯 What you need:

✓ Python 3.7+
✓ FFmpeg (brew install ffmpeg on macOS)
✓ Gemini API key (free tier available)
✓ Noto Sans Bengali font (pre-installed on macOS)


🚀 Quick Test

  ./Scripts/workflow_example.sh

💡 Tips

• Create new project for each podcast: mkdir -p "Projects/Name"/{Source,Transcripts,Analysis,Processing,Output}
• Each project is self-contained and organized
• Final reels saved in: Projects/ProjectName/Output/
• Use Bengali titles with underscores: আপনার_শিরোনাম.mkv
• Each reel takes ~5-10 minutes to create
• Gemini API cost: ~$0.90 per hour of podcast


⚠️ Important

• ALWAYS use ASS format for Bengali text (not drawtext)
• Ensure FFmpeg has libass support for proper Bengali rendering
• Keep your Gemini API key secure
