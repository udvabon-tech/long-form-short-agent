### Scripts

**transcribe_audio_v2.py**
- Uses Google Gemini 2.5 Pro API
- Transcribes Bengali audio
- Generates detailed timestamps [HH:MM:SS.mmm]
- Identifies speakers automatically
- Usage: `python3 Scripts/transcribe_audio_v2.py "Projects/$PROJECT/Transcripts/audio.mp3"`

**analyze_transcript.py**
- Analyzes transcripts for viral moments
- Uses Gemini AI to identify best segments
- Generates cut directions with timestamps
- Creates comprehensive reel analysis
- Usage: `python3 Scripts/analyze_transcript.py`

**create_timestamp_text.py**
- Extracts specific time ranges from transcript
- Creates word-level timestamps
- Adjusts timestamps to start from 0
- Estimates 400ms per word
- Edit script to point to your project's files

**text_to_enhanced_ass.py**
- Converts word timestamps to ASS format
- Adds karaoke highlighting effects
- Uses Noto Sans Bengali font (75pt)
- Groups words into 2-3 word subtitle blocks
- Every 4th word highlighted in red
- Usage: `python3 Scripts/text_to_enhanced_ass.py Processing/input.txt Processing/output.ass`

**workflow_example.sh**
- Automated end-to-end workflow
- Edit configuration variables at top
- Runs all steps sequentially
- Progress indicators and error handling
- Optional cleanup at end
- Usage: `./Scripts/workflow_example.sh`

## 🚀 Quick Start

1. **Install prerequisites:**
   ```bash
   pip install -r requirements.txt
   export GEMINI_API_KEY="your-key-here"
   ```

2. **Create a project:**
   ```bash
   PROJECT="My_Podcast_Name"
   mkdir -p "Projects/$PROJECT"/{Source,Transcripts,Analysis,Processing,Output}
   cp ~/path/to/video.mp4 "Projects/$PROJECT/Source/"
   ```

3. **Read the guides:**
   - First time: Start with `QUICKSTART.md`
   - Organization: Check `FILE_ORGANIZATION.md`
   - Need details: See `README.md`
   - Advanced usage: Read `AGENT.md`

4. **Run the workflow:**
   - Manual: Follow steps in `QUICKSTART.md`
   - Automated: Edit and run `Scripts/workflow_example.sh`

## ⚠️ Important Notes

### Bengali Text Rendering
- Always use ASS subtitles for Bengali text
- Never use FFmpeg `drawtext` filter (breaks Bengali characters)
- ASS filter uses libass + HarfBuzz for proper text shaping

### File Naming
- Use Bengali titles with underscores: `আপনার_শিরোনাম.mkv`
- Avoid spaces in filenames (causes command-line issues)

### API Usage
- Gemini API transcription costs: ~$0.00025 per second of audio
- 1-hour podcast ≈ $0.90
- Keep API key secure, never commit to git
