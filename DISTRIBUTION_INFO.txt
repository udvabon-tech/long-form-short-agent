⚠️ IMPORTANT NOTES

Bengali Text Rendering:
  • Always use ASS subtitles (not drawtext)
  • drawtext breaks Bengali characters
  • ASS uses libass + HarfBuzz for proper shaping

File Naming:
  • Use Bengali titles with underscores
  • Example: আপনার_শিরোনাম.mkv
  • Avoid spaces in filenames

API Security:
  • Keep your API key secure
  • Never commit to version control
  • Use environment variables

🔍 TROUBLESHOOTING

Issue: API key not working
  → Verify: echo $GEMINI_API_KEY
  → Reset: export GEMINI_API_KEY="key"

Issue: Broken Bengali text
  → Using drawtext? Switch to ASS
  → Check AGENT.md for explanation

Issue: FFmpeg errors
  → Verify libass: ffmpeg -version
  → Reinstall: brew reinstall ffmpeg

Issue: Font not found
  → macOS: Pre-installed
  → Linux: apt-get install fonts-noto-bengali
  → Windows: Download from Google Fonts
